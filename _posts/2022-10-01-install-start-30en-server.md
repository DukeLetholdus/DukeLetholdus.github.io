# Start the 30en Server

## 0. Dependancies

first install node.js:
```bash
sudo apt install nodejs
```

Install Node Package Manager to add packages:
```bash
sudo apt install npm
```
To check if node and npm installed correctly:
```bash
node -v
```
```bash
npm -v
```
